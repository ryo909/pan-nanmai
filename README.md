# パン何枚ったー

今までに食べたパンの枚数をフェルミ推定で概算し、これから食べるぶんもワンタップで追加できる静的Webアプリです。

## 公開URL

GitHub Pages 公開後:

- https://ryo909.github.io/pan-nanmai/

## 構成

- `index.html`: エントリーポイント
- `styles.css`: 共通トークンと全画面スタイル
- `app.js`: 画面描画、フェルミ推定ロジック、localStorage 保存、共有処理

## ローカル起動

```bash
python3 -m http.server 4173
```

`http://127.0.0.1:4173/` を開いて確認できます。

## 機能

- 年齢に応じて質問数が変わるスライド型質問フロー
- 年代別のフェルミ推定と誤差レンジ表示
- 結果コメントとシェア文生成
- 未来追加カウントと簡易履歴
- localStorage による再訪維持
- 設定 / リセット画面
